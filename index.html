Hacer un juego

Para hacer un juego mínimo, crea una carpeta en cualquier lugar y abre tu editor de texto favorito. Sublime Text es bastante bueno para todos los sistemas operativos, y tiene soporte Lua incorporado. Cree un nuevo archivo en la carpeta que acaba de crear y asígnele el nombre . Coloque el código siguiente en el archivo y guárdelo. main.lua

function love.draw()

love.graphics.print("Hello World", 400, 300)

end

LÖVE puede cargar un juego de dos maneras:

  • Desde una carpeta que contiene un archivo main.lua.
  • Desde un archivo .love que tiene un archivo main.lua en el nivel de directorio superior (también conocido como root)
Para crear archivos .love, consulte Distribución de juegos.

Echa un vistazo a este código:

print("I like the number " .. num)

Dentro de los paréntesis de la función podemos poner lo que llamamos parámetros. Los parámetros son variables temporales que solo existen dentro de la función. En este caso colocamos el parámetro . Y ahora podemos usar como cualquier otra variable.numnum

Devolución

Las funciones pueden devolver un valor, que podemos almacenar dentro de una variable, por ejemplo. Puede devolver un valor mediante la palabra clave.return

function giveMeFive()

return 5

end

a = giveMeFive()

print(a)

--Output: 5

a se convierte en el valor que giveMeFive devuelve.

Resumen

Las funciones pueden almacenar código que podemos ejecutar en cualquier momento. Llamamos a una función escribiendo su nombre seguido de paréntesis. Podemos poner valores dentro de estos paréntesis. Estos valores se pasan a los parámetros de la función, que son variables temporales que solo existen dentro de la función. Las funciones también pueden devolver un valor. Las funciones eliminan la repetición y eso es algo bueno.

¿Qué es LÖVE?

LÖVE es lo que llamamos un marco. Dicho simplemente: Es una herramienta que facilita la programación de juegos. LÖVE está hecho con C++ y OpenGL,que se consideran muy difíciles. El código fuente de LÖVE es muy complejo. Pero todo este código complejo hace que crear un juego sea mucho más fácil para nosotros.

Lua

Lua es el lenguaje de programación que utiliza LÖVE. Lua es un lenguaje de programación por sí solo, y no está hecho por o para LÖVE. Los creadores de LÖVE simplemente eligieron Lua como el lenguaje para su marco. Entonces, ¿qué parte de lo que codificamos es LÖVE y qué es Lua? Muy sencillo: Todo lo que empieza por forma parte del framework LÖVE. Todo lo demás es Lua.love. Estas son funciones que forman parte del marco LÖVE:

¿Cómo mente LÖVE?

LÖVE llama a 3 funciones. Primero llama a love.load(). Aquí creamos nuestras variables. Después de eso, llama a love.update() y love.draw(), repetidamente en ese orden. Así: love.load -> love.update -> love.draw -> love.update -> love.draw -> love.update, etc. Detrás de las escenas, LÖVE llama a estas funciones, y nosotros para crearlas, y llenarlas con código.
